# 로블록스 시장 분석 봇(Bot): 기술 명세서

본 모듈은 로블록스의 분산 마이크로서비스 백엔드 엔드포인트를 타겟팅하여 인게임 경제 시스템과 리텐션 메커니즘을 프로그래밍 방식으로 추출하고, '개발 3대 헌법'에 맞춘 시장 분석 보고서를 자동 빌드하는 데이터 파이프라인입니다.

## 🏗️ 시스템 아키텍처 작동 원리

본 봇은 **[데이터 수집 ➔ 정제 및 연산 ➔ 마크다운 자동화]**의 3단계 파이프라인으로 작동합니다.

### 1. 다중 엔드포인트 수집 프로세스
* **`games.roblox.com`**: 실시간 동시 접속자 수(CCU), 누적 방문자 수, 유저들의 업보트/다운보트 원천 수치를 수집합니다.
* **`economy.roblox.com`**: 유료 게임패스의 명칭과 가격(`price`) 데이터를 수집하여, 유저가 마주하는 결제 유도 포인트를 데이터로 치환합니다.
* **`badges.roblox.com`**: 업적 시스템의 지난주 획득 횟수(`pastWeekAwardedCount`) 지표를 추적하여 유저가 실제로 어디까지 도달하고 탈락하는지(리텐션 깔대기 모델)를 직접 플레이 없이 역산합니다.

### 2. 데이터 가공 및 분석 연산
* **유저 평점 수식 연산**: 원천 데이터인 `upVotes`와 `downVotes`를 가공하여 실시간 승인율 지표를 산출합니다.
  $$\text{Upvote Ratio} = \frac{\text{upVotes}}{\text{upVotes} + \text{downVotes}} \times 100$$
* **3대 헌법 자동 매칭**: 산출된 데이터 평점 및 CCU 볼륨을 규칙 기반(Rule-based) 알고리즘에 대입하여, 목표 체류 시간(9~15분) 달성 여부 및 BM 고도화 상태를 추론 텍스트로 자동 바인딩합니다.

### 3. 서버 레이트 리밋(Rate Limit) 우회
* IP 차단 및 트래픽 과부하를 방지하기 위해 요청 사이에 의도적인 딜레이 타임(`time.sleep`)을 배치하여 안정적인 대량의 데이터 수집 환경을 구축했습니다.
